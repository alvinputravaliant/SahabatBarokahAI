# 🌙 SahabatBarokahAI

**SahabatBarokahAI** is a simple AI-powered web platform that helps small businesses in Indonesia generate ideas and strategies for the Ramadan season.

The platform uses AI to help users quickly create:

* Ramadan business strategies
* viral menu ideas
* product profit analysis
* marketing content

Everything is available in one simple dashboard.

---

## 🚀 Features

**AI Strategy Generator**
Generate Ramadan business strategies based on business type, target market, and budget.

**Viral Menu Generator**
Create creative food or drink menu ideas that could attract more customers.

**Profit Calculator AI**
Estimate product profit, daily revenue, and potential profit during Ramadan.

**Marketing Content Generator**
Generate marketing ideas, captions, and content concepts for social media.

---

## 🧠 Tech Stack

This project is built using:

* Next.js
* PostgreSQL
* Prisma
* OpenAI API
* Tailwind CSS
